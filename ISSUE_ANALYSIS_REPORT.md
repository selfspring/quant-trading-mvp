### 1. 模拟信号导致重复开仓
**问题**：
```python

    "prediction": 0.015,
    "confidence": 0.75,
    "signal": 1  # 始终看多

```

**结果**：
- 每次循环都生成"买入开仓"意图
- 风控检查发现持仓为 0，放行开仓
- 系统不断发送重复订单

### 2. 订单状态码 5 的含义
**观察**：
```
[报单回报] 买 au2606 1手 @0.0 状态=a Ref=X  ← 已接受
[报单回报] 买 au2606 1手 @0.0 状态=5 Ref=X  ← 状态变化
```

**可能的原因**：
- `5` = 已撤单 (Cancelled)
- `5` = 废单 (Rejected)
- `5` = 部分成交后撤单

**推测**：
- 市价单在夜盘可能因为流动性不足被撤单
- 或者 SimNow 对市价单有特殊处理

### 3. 持仓同步问题
**观察**：
- 每 5 个循环同步一次持仓

#### 2. 添加订单去重逻辑
**在 `TradeExecutor` 中添加**：
```python
class TradeExecutor:
    def __init__(self, config, td_api):
        self.last_order_time = None
        self.min_order_interval = 300  # 5分钟内不重复发单
    
    def execute_order(self, order_intent):
        # 检查是否在冷却期
        if self.last_order_time:
            elapsed = (datetime.now() - self.last_order_time).total_seconds()
            if elapsed < self.min_order_interval:
                logger.warning(f"订单冷却中，距离上次发单 {elapsed:.0f} 秒")
                return
        
        # 发单逻辑...
        self.last_order_time = datetime.now()
```
